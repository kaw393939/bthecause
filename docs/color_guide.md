# Theoforge - Color Guide (v1.0)\n\n## 1. Color Philosophy & Strategy\n\nColor is a primary vehicle for communicating Theoforge's brand identity as The Sage: knowledgeable, trustworthy, guiding, clear, and structured. Our color strategy aims to build trust, convey expertise, ensure clarity, and create a professional, sophisticated visual experience appropriate for engaging senior executives (CEO/CIO) in Fortune 100 companies.\n\nKey Strategic Principles:\n\n*   **Trust & Stability:** Establish a foundation of trust through a stable, authoritative primary color.\n*   **Clarity & Structure:** Utilize a clear neutral palette for optimal readability and visual organization.\n*   **Sophistication & Quality:** Employ secondary colors subtly to signal premium value and expertise.\n*   **Focused Action:** Use accent colors sparingly and purposefully to guide users towards key actions.\n*   **Accessibility:** Ensure all color combinations meet or exceed WCAG AA contrast requirements, demonstrating consideration and professionalism.\n\n## 2. Core Palette Definition\n\nThis palette is implemented in `/src/theme/theme.ts`.\n\n### 2.1. Primary Color: Deep Teal\n\n*   **Role:** Core brand identifier, trust, stability, intelligence, primary actions (buttons, links).\n*   **Definition:**\n    *   `palette.primary.main`: `#00695C` (Teal 700)\n    *   `palette.primary.light`: `#439889`\n    *   `palette.primary.dark`: `#003D33`\n    *   `palette.primary.contrastText`: `#FFFFFF`\n\n### 2.2. Secondary Color: Muted Gold\n\n*   **Role:** Subtle highlighting of quality, value, wisdom, secondary accents (e.g., testimonials, specific callouts). Use sparingly.\n*   **Definition:**\n    *   `palette.secondary.main`: `#B8860B` (DarkGoldenrod)\n    *   `palette.secondary.light`: `#E6A634`\n    *   `palette.secondary.dark`: `#8A5F00`\n    *   `palette.secondary.contrastText`: `#FFFFFF` (Ensure sufficient contrast in application)\n\n### 2.3. Neutrals\n\n*   **Role:** Foundation for text, backgrounds, structure, ensuring clarity and readability.\n*   **Text:**\n    *   `palette.text.primary`: `#212121` (Grey 900)\n    *   `palette.text.secondary`: `#616161` (Grey 700)\n    *   `palette.text.disabled`: `#9E9E9E` (Grey 500)\n*   **Backgrounds:**\n    *   `palette.background.default`: `#FFFFFF`\n    *   `palette.background.paper`: `#FFFFFF`\n*   **Dividers:**\n    *   `palette.divider`: `#E0E0E0` (Grey 300)\n*   **Full Scale:** `palette.grey` provides access to the full MUI grey scale if needed for specific UI elements.\n\n### 2.4. Accent Color: Bright Orange\n\n*   **Role:** Highlighting primary Calls to Action (CTAs), drawing focused attention. **Use with extreme restraint.** Overuse will diminish impact and appear unprofessional.\n*   **Definition (Currently uses `warning` slot):**\n    *   `palette.warning.main`: `#F57C00` (Orange 700)\n    *   `palette.warning.light`: `#FFAD42`\n    *   `palette.warning.dark`: `#BB4D00`\n    *   `palette.warning.contrastText`: `#FFFFFF`\n*   **Strategic Recommendation:** Consider defining this as a dedicated `accent` color in the theme (e.g., `palette.accent.main`) rather than using the `warning` slot for primary CTAs. This improves semantic clarity and avoids unintended associations.\n\n### 2.5. System Feedback Colors\n\n*   **Role:** Standard UI feedback (errors, information, success).\n*   **Definitions:**\n    *   `palette.error.main`: `#D32F2F` (Red 700)\n    *   `palette.info.main`: `#0288D1` (Light Blue 700)\n    *   `palette.success.main`: `#2E7D32` (Green 700)\n    *   *All use `#FFFFFF` for `contrastText`.*\n\n## 3. Color Usage Guidelines\n\n*   **Dominance:** The primary palette (Teal and Neutrals) should dominate the visual experience, establishing the core brand feel.\n*   **Secondary Accentuation:** Use Muted Gold (`secondary.main`) sparingly for specific, high-value highlights. Avoid using it for general interactive elements.\n*   **CTA Hierarchy:** Reserve Bright Orange (`warning.main` / proposed `accent.main`) exclusively for the most critical conversion points.\ Less critical actions should use the primary Teal.\n*   **Text Hierarchy:** Use `text.primary` for main content and headings. Use `text.secondary` for supporting information, labels, and metadata where appropriate contrast is ensured.\n*   **Consistency:** Strictly adhere to the theme palette. Avoid hardcoding hex values in components. Use theme variables (`theme.palette.primary.main`, `sx={{ color: 'secondary.main' }}`).\n\n## 4. Accessibility & Contrast\n\n*   **Minimum Standard:** All text/background combinations *must* meet WCAG 2.1 Level AA contrast ratios.\n*   **Target Audience Consideration:** For critical text content aimed at potentially older executives, strive for Level AAA compliance where design permits, particularly for `text.secondary`.\n*   **Tooling:** Utilize contrast checking tools (browser extensions, online checkers) during design and development.\n*   **Verification Needed:** Specifically verify contrast for:\n    *   `text.secondary` (`#616161`) on `background.paper` (`#FFFFFF`) - *Ratio is 4.68:1 (AA Pass, just above minimum)*. Consider `grey[800]` (`#424242`, 7.26:1) for better comfort if needed.\n    *   `contrastText` (`#FFFFFF`) on `secondary.main` (`#B8860B`) - *Ratio is 3.9:1 (Fail AA for normal text, Pass AA Large)*. Requires careful usage - only for large text (18pt+) or bold text (14pt+), or consider dark contrast text.\n    *   `contrastText` (`#FFFFFF`) on `warning.main` (`#F57C00`) - *Ratio is 3.05:1 (Fail AA for normal text, Pass AA Large)*. Same constraints as Muted Gold apply.\n\n## 5. Implementation Notes\n\n*   Source of Truth: `/src/theme/theme.ts`\n*   Usage: Leverage Material UI theme structure (`theme.palette...`).\n\n---\n*Version 1.0 - Based on theme.ts verification and brand_guide.md v0.2 strategy.*\n
